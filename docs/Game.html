Game

Contains/controls primary game components as well as currently active simulation.
Properties:
Name Type Description
start function Starts a new game.
running boolean Whether system is currently being simulated or not.
gui dat.GUI Null to start, will contain a GUI for the game once it has loaded up.
generateNewSystem function Generates a new system.
temporaryShittyShip function Creates a basic Ship that works right now but will probably break in the future and is already slightly broken and temporary overall anyhow.
generateAsteroids function Generates some asteroids in the system randomly.
loop function Main simulation loop, applies velocity, gravity, checks for input, renders.
